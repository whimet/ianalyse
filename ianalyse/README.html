Install Ianalyse

Install Python

Download the python installer from http://www.python.org/ftp/python/2.6.4/, install it on the default location, put python into your PATH, to verify it is installed correctly, start up the console and type:

$ python --version
You should be able to see the output:
$ Python 2.6.4
Note: I strongly recommended you to use python 2.6, but it is not necessary, you can use python 2.5.4 as the runtime environment, however, you need to ensure python-cjson was installed correctly. if you have easy_install installed, you can easily install the python-cjson by typing:
$ easy_install python-cjson=1.0.3
Read this blog if you got any problem: http://blog.sadphaeton.com/2009/01/20/python-development-windows-part-2-installing-easyinstallcould-be-easier.html

Install Django

Visit http://www.djangoproject.com/download/ to download and install Django framework, follow the steps on that page.

Start the application

Depends on the operation system you are using, run production_server.bat or production_server.sh accordingly. input your non-lookback ip address (run ifconfig or ipconfig to get this information) when prompting: 

$ ./production_server.sh 
please type the non-loopback(not 127.0.0.1 or 0.0.0.0) ip address of your machine:
192.168.1.94
if the server starts up correctly, you should be able to see the output like those: 
$ ./production_server.sh 
please type the non-loopback(not 127.0.0.1 or 0.0.0.0) ip address of your machine:
192.168.1.94
Validating models...
0 errors found

Django version 1.1, using settings 'ianalyse.settings'
Development server is running at http://192.168.1.94:8000/
Quit the server with CONTROL-C.

By default ianalyse will listen to the port 8000, you can change this by editing the file production_server.bat or production_server.sh(depends on the operation system you are using)

Using the application

Read CONF.html to understand how to configure the ianaylse, To use the application, open up your browser and visit the http://IP Address:8000/
